Expert System for Disease Diagnosis 🩺🤖

An AI-powered medical expert system developed using Python and Prolog for intelligent disease diagnosis based on patient symptoms. The system mimics human expert reasoning using a rule-based inference engine and logical knowledge representation.

This project demonstrates the integration between Artificial Intelligence and Logic Programming to build a smart healthcare assistant capable of predicting possible diseases and providing basic medical recommendations.

🚀 Features
Symptom-based disease prediction
Rule-based expert system
Knowledge base implemented in Prolog
Interactive Python interface
Logical reasoning and inference engine
Easy to extend with new diseases and symptoms
Explainable AI decisions
🧠 Technologies Used
Python
Prolog
PySWIP
Artificial Intelligence
Expert Systems
Logic Programming
📌 System Workflow
User enters symptoms through the Python interface.
Python communicates with the Prolog knowledge base.
Prolog applies logical rules and inference.
The system predicts the most probable disease.
Diagnosis results and recommendations are displayed.
